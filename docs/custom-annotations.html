How to Customize for Your Own Code

You can easily adapt this approach for your own code by:

  1. Copy the CSS styles into your document
  2. Use the HTML structure shown in the examples:
    • Create a .code-with-annotations container
    • Add your code inside a .code-block div
    • Put each line in a .code-line div
    • Add annotation markers where needed
    • Position annotation bubbles with appropriate top values

Alternative: Interactive Annotations
